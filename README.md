## 📱 About This Software

This is a **mobile-ready web application** that helps users calculate the cost and specifications for a complete renewable energy system including:

- ☀️ **Solar PV** (Monocrystalline panels)
- 🔋 **Sodium-ion Battery Storage** (Safe, affordable, cold-weather friendly)
- 💨 **Optional Wind Turbine Backup**
- 🏠 **Home Backup** (Basic or Pro tiers)
- 🔮 **Future Development Ready** (Hydrogen, Hydro, EV charging)

## 🎯 Who Is This For?

| Tier | Description |
